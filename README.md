👤 Profile Card UI

A simple responsive profile card built using HTML and CSS.

📌 Features

- Clean modern design
- Profile image section
- Name, role, and social links section
- Hover effects

🛠️ Tech Used

- HTML5
- CSS3
- Google Fonts

🎯 Purpose

Practice basic layout design using Flexbox and card UI structure.
